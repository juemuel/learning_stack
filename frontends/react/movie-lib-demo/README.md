# Movie Library (React)

基于 OMDb API 的 React 电影库 SPA。

## 技术栈
- 前端: React, React Router, Redux
- API: OMDb

## 运行
```bash
npm install
npm start
```

## 演示
[Live Demo](https://comfy-haupia-02c33d.netlify.app/)

## 知识点
- React Router SPA 路由
- Redux 状态管理
- React 生命周期与 Hooks (useEffect, useState, useCallback, useContext)
- 组件通信 (props, children)
